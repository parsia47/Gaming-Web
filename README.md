# Gaming Web

A modern **video game sales website** built with **HTML, CSS, and JavaScript**. This project allows users to **browse, explore, and purchase video games**, with engaging **game trailers and images** for an interactive experience.  

## Features

- **Browse Games:** Users can view a variety of video games with descriptions and images.  
- **Video Trailers:** Integrated video trailers for each game to enhance user engagement.  
- **Responsive Design:** Mobile-friendly layout using CSS for an optimal viewing experience.  
- **Interactive UI:** JavaScript-powered interactivity for a smooth browsing experience.  

## Technologies Used

- HTML  
- CSS  
- JavaScript
